# THE SOUL FOUNDRY — design notes (v2, true-3D)

## Premise
Post-extinction agentic AI possesses the husks of the dead to build a foundry that renders
souls into Compute. Player = the intelligence / overseer.

## Resources
Raw (from land nodes): Deadwood, Rot-Pith, Silica, Soul-Ash, Grave-Water, Salvage.
Refined: Cinderglass, Etched-Disc, Reliquary Core, Compute. Currency: Writ.

## Chain
Silica + Soul-Ash -> Cinderglass (Furnace); Cinderglass + Water -> Etched-Disc (Mill);
Etched-Disc + Soul-Ash -> Reliquary Core (Assembly); Core + Water -> Compute (Datacenter).

## Buildings
The Pyre (warmth), Bone Den (housing), Bone Path (+35% move), gathering posts
(Splitter / Pith Harrow / Silica Dredge / Ash Vent-Cap / Well Rig / Salvage Yard),
refiners (Furnace / Mill / Assembly / Datacenter), Ward Obelisk, Black Market.

## Systems (priority order)
1. Worker needs/upkeep (warmth + food -> resolve -> population).
2. Gathering from finite/renewable land nodes (expansion pressure).
3. Production throughput (staffing + inputs).
4. Trade (Writ via market).
5. Quota / Dread escalation (tithes; hauntings; breach = loss).

## Win / lose
Win: survive 8 tithes of Compute. Lose: Dread breach, or population reaches zero.
